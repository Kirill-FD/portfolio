# Инструкция по запуску проекта

Одностраничный сайт-портфолио на **React**, **TypeScript** и **Vite**.

## Требования

- **Node.js** 18+ (рекомендуется 20 LTS)
- **npm** 9+ (или pnpm/yarn)

## Установка зависимостей

В корне проекта выполните:

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

После запуска откройте в браузере адрес, который выведет Vite (обычно **http://localhost:5173**).

## Сборка для продакшена

```bash
npm run build
```

Результат сборки будет в папке **`dist/`**. Её можно развернуть на любом статическом хостинге (Vercel, Netlify, GitHub Pages и т.п.).

Предпросмотр собранной версии локально:

```bash
npx serve dist
```

## Проверка кода

- **Проверка типов TypeScript:**
  ```bash
  npm run typecheck
  ```

- **Проверка линтером ESLint:**
  ```bash
  npm run lint
  ```

## Доступные скрипты

| Команда           | Описание                          |
|-------------------|-----------------------------------|
| `npm run dev`     | Запуск dev-сервера с hot reload   |
| `npm run build`   | Сборка для продакшена в `dist/`   |
| `npm run typecheck` | Проверка типов (tsc --noEmit)   |
| `npm run lint`    | Запуск ESLint для папки `src`     |

## Структура проекта

- `src/` — исходный код приложения
  - `src/app/` — компоненты и разметка приложения
  - `src/styles/` — глобальные стили
- `index.html` — точка входа HTML
- `vite.config.ts` — конфигурация Vite
- `tsconfig.json` — конфигурация TypeScript

## Устранение неполадок

1. **Ошибки при `npm install`**  
   Убедитесь, что версия Node.js не ниже 18. Проверка: `node -v`.

2. **Порт 5173 занят**  
   Vite автоматически предложит следующий свободный порт (например, 5174).

3. **Ошибки типов или линтера**  
   Выполните `npm run typecheck` и `npm run lint` и исправьте сообщения в консоли.
